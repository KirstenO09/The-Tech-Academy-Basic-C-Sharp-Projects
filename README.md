# 💻 C# and .NET Projects – The Tech Academy

This repository contains a collection of projects completed during the **Tech Academy Software Developer Bootcamp**.
The projects focus on learning **C# programming fundamentals, object-oriented programming, and .NET application development**.

Some folders contain **larger projects**, while others contain **short programming exercises ("Assignments")** designed to practice specific programming concepts.

Together, these projects demonstrate the development of core software engineering skills using the **Microsoft .NET ecosystem**.

# ⭐ Featured Projects

## 🎰 Casino Console Application (Favorite Project)

The **Casino** project is my favorite console application in this repository.
It simulates a simple casino environment where a player can interact with the system and play a game based on logic implemented in C#.

This project demonstrates several important programming concepts including:

* Object-Oriented Programming
* Class design and inheritance
* Program flow control
* Exception handling
* Lists and collections
* File logging

The project is structured like a real C# application with multiple classes and organized logic rather than a single script.

---

## 🚗 Car Insurance Web Application

The **CarInsurance** project is an ASP.NET MVC web application that calculates insurance quotes based on user input.

Users enter information such as:

* Age
* Car year
* Car make and model
* Driving history

The application processes the data and calculates a monthly insurance quote using business rules implemented in C#.

Concepts demonstrated:

* ASP.NET MVC architecture
* Controllers, Views, and Models
* Entity Framework
* Form handling
* Business logic implementation
* Database interaction

This project demonstrates how C# is used in **web development environments**.
